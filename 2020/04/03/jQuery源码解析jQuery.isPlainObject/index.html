jQuery.isPlainObject 源码解析

定义和用法

$.isPlainObject() 函数用于判断指定参数是否是一个纯粹的对象。

注意：宿主对象(或其它被浏览器宿主环境使用的对象，用来完成ECMAScript的执行环境)难以进行跨平台的特性检测。因此，对于这些对象的实例，$.isPlainObject()在不同的浏览器上可能得出不同的结果。

语法

$.isPlainObject( object ) 参数 object ：任意类型 需要进行判断的任意值。

详细使用，可参考菜鸟教程：https://www.runoob.com/jquery/misc-isplainobject.html
